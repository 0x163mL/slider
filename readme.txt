
        Start with any file in folder 'themes', 'examples'
        e.g. you can start with 'examples/simple-slider.source.html'.
        Use 'jssor.core.js' 'jssor.utils.js' 'jssor.slider.js' for developement.
        Use 'jssor.slider.mini.js' (30KB for jQuery Plugin) or 'jssor.slider.min.js' (60KB for No-jQuery Version) for release.
        In addition you can run 'examples/simple-slider.compress.bat' to compress your slider to code snippet, see 'examples/simple-slider.html'.
        Reference http://www.jssor.com/developement/index.html


        Note that 'jssor.compress' runs on windows system with .NET 2.0 installed.
